# vue3-init
## 基于vue-cli创建vue3+TS项目步骤
1. npm install -g @vue/cli
2. vue --version // 4.5.0以上才行
3. vue create my-project
4. 选第三个手动选择->选TS按下空格->选vue3->一路回车
